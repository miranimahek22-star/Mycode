# Mycode
#spi and cpi calculator
FEATURES:
* Asks for the number of subjects in a semester,total credits and grade obtained to calculate spi for each semester.
* After calculating spi for both semesters automatically calculates cpi for the year.
* Determines your academic standing and calculate the required points to be in 10% of your class.
* If the passing criteria of any semester is not fulfilled then even if the overall cpi is greater than the required one it will show fail.
